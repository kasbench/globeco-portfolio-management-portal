# Proposed Tech Stack

**Frontend Framework: Next.js with React**
- Next.js provides excellent developer experience with built-in routing, API routes, and deployment optimization
- React has the largest community and best AI assistance
- TypeScript support out of the box for better code quality

**Styling: Tailwind CSS**
- Utility-first approach that's beginner-friendly
- No need to write custom CSS
- Excellent integration with modern React components

**UI Components: shadcn/ui**
- Pre-built, customizable components
- Built on Radix UI primitives for accessibility
- Works seamlessly with Tailwind CSS

**State Management: React Query (TanStack Query) + Zustand**
- React Query handles server state and API calls beautifully
- Zustand for simple client-side state when needed
- Both are lightweight and beginner-friendly

**HTTP Client: Axios or native fetch**
- Axios for more complex API interactions
- Native fetch for simpler calls

**Development Tools:**
- Vite or Next.js dev server for fast development
- ESLint and Prettier for code quality
- TypeScript for catching errors early
